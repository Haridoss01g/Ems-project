#Employee Management System(EMS)
This is a Django-based web application for managing employees.

##project Description
This project helps to managee employee details like adding,deleting, and viewing employee records in an easy way.

## features
Add Employee
Edit Employee
Delete Employee
View Employee List
Search Employee
Dashboard

##Tech Stack
Python
Django Framework
HTML
CSS
SQLite Database
using javascript(minimal)

##
Purpose
To simplyfy employee data management in organizations.

## How to run Project
1.Clone the responsitory
2.Install Djano:
pip install django
3.Run server:
python manage.py runserver
4.open browser:
http://127.0.0.1:8000/
